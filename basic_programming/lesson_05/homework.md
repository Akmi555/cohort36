# Задача 1

По аналогии с задачей 5 из классной работы (файл `Task5Watches.java`) напишите следующую программу:

Вводят часы, минуты и секунды.
Нужно вывести общее количество секунд с начала суток.

## Пример ввода
```
9 45 37
```

## Пример вывода
```
35137
```
