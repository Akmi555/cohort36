# Задание

* `Main01`
Сложность: легко (2/10)
```
Спросить у пользователя номер дня недели и вывести название этого дня недели в консоль.
Программа не должна принимать числа, которые находятся не в диапазоне от 1 до 7.
Для реализации превращения числа в название написать метод, в котором используется switch
Пример:
  Ввод:
  4
  Вывод:
  Четверг
```

* `Main02`
  Сложность: средне (4/10)
```
Спросить у пользователя название месяца. Вывести количество дней в этом месяце.
Для этого задания не учитывать высокосный год - считаем что в феврале всегда 28 дней.
Так же как и в предыдущем задании, использовать метод и switch
Пример:
  Ввод:
  Октябрь
  Вывод:
  31
```
