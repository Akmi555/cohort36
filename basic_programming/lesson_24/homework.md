# Задание

```
Создать проект с названием homework и инициализировать в нем новый репозиторий.
Создать репозиторий с таким же названием в своем гитхабе, добавить преподавателя в коллабораторы
Добавить этот репозиторий как origin в проект homework.
Создать .gitignore (написать в него все что там должно быть) и README.md,
добавить в трек и создать коммит с сообщением init.
В файле README.md написать, кто является автором репозитория, и что это репозиторий для обучения Basic Java
Запушить master в ремоут.
Создать и переходим в ветку с названием homework24. Создать пакет homework24, в нем файл Main01.
В этом файле написать программу Hello world.
Сохранить в коммит этот пакет.
Запушить эту ветку в ремоут и создать пулл реквест. Отправить ссылку на пул реквест преподавателю.
После одбрения пул реквеста, перейти на ветку master, сделать pull для ветки main и удалить ветку homework24.
```
