# Задание

* `Main01`
Сложность: легко (2/10)
```
Написать методы для рассчета суммы, разницы, произведения и частного.
В методе с делением учесть случай деления на ноль. Если пользователь пытается поделить на ноль, нужно написать в консоль
что делить на ноль нельзя и в качестве результата вернуть ноль.
В программе, попросить пользователя ввести два числа, и вывести результат всех операций над этими числами.
Для операций использовать написанные ранее методы.
Пример:
    Ввод:
    15 2
    Вывод:
    15 + 2 = 17
    15 - 2 = 13
    15 * 2 = 30
    15 / 2 = 7.5
```

* `Main02`
  Сложность: средне (4/10)
```
Написать методы для анализа числа на парность, знак, целость и простоту.
В программе, считать одно число типа double и для него вызвать все написанные методы
Подсказки к алгоритмам:
Парность - если число делится на 2 без остатка. Возможные результаты: четное, нечетное
Знак - сравнить число с нулем. Возможные результаты: положительное, отрицательное, ноль
Целость - сравнить число и его приведенный к целочисленному вариант. Возможные результаты: натуральное, целое, нецелое
Простота - если число не делится нацело на все числа от 2 до этого числа минус один.
Подсказка к подсказке: в последнем использовать цикл
Пример 1:
    Ввод:
    -15,2
    Вывод:
    Число нечетное
    Число отрицательное
    Число нецелое
    Число не простое
Пример 2:
    Ввод:
    7
    Вывод:
    Число нечетное
    Число положительное
    Число натуральное
    Число простое
```
