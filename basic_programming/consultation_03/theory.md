### 01. можно ещё раз объяснить, откуда берётся toCharArray
* toCharArray - это метод, который позволяет преобразовать String в массив типа char
* Берется он, довольно логично, из класса String

### 02. простые числа, это натуральные числа, которые делятся на цело на 1 и само на себя. но мы писали другой код. я тут запуталась
* Простое число - это такое натуральное число, которое делится нацело только на себя и на единицу
* Натуральное число - это такое целое число, которое используется при счете
* Целое число - это такое число, которое при делении на 1 не дает остатка (не имеет нецелой части)

### 04. Суммировать все изученное пока что в теме гит
* При разработке приложения с гитом, у нас всегда есть определенная последовательность действий.
1. С самого начала, нужно создать либо склонировать репозиторий.
2. Если репозиторий пустой, создать изначальный (initial) коммит.
3. При разработке, когда нужно, делать "чекпоинты" в своей программе, к которым можно, в случае поломки, вернуться
4. Для того чтобы это сделать, нужно сначала добавить все файлы в отслеживание (track). Добавлять стоит только те файлы,
которые вы хотите сохранять. После этого, нужно выполнить сохранение изменений (commit changes).
5. Повторять действия 3-4 пока программа не будет готова к отправке в ремоут.
6. Когда программа (класс/фича/функция/метод) готова, отправить в ремоут.
7. Потом, если нужно, получить изменения с ремоута и продолжить работу с пункта 3.

### 05. Метод преобразования массива в строку. Тайпкасты
* Массив можно преобразовывать в строку с помощью Arrays.toString().
* При этом все элементы массива будут выведены через запятую, и взяты в квадратные скобки.

* Type cast (преобразование типа) - это когда мы перед выражением либо переменной ставим в скобках нужный нам тип,
чтобы преобразовать значение выражения или переменной в этот тип.
