# Задание

* `Main01`
Сложность: сложно (7/10)
```
"Виселица"

Воссоздать популярную школьную игру "Виселица", где один человек загадывает слово, а второй должен отгадать слово по буквам,
имея шанс сказать неправильную букву только 7 раз.

При запуске, программа должна выбрать одно случайное слово из списка возможных слов.
Далее пользователю должно быть представлена строчка, в которой по одному подчеркиванию для каждой буквы этого слова.
Пользователь должен ввести любую букву.
Если буква есть в слове, то вместо подчеркивания теперь нужно выводить эту букву. Если буквы нет, отнять одну жизнь (из 7)
Если пользователь успешно угадал слово, поздравить с выигрышем и закончить програму.
Если у пользователя закончились жизни, написать что он проиграл, написать какое было слово и закончить программу.

Ниже будут приведены направляющие подсказки к этому заданию. Это не точное руководство, вы можете делать и по другому.
Подсказки к выполнению задания:
- Для выбора слова в программе нужно создать массив строк со всеми возможными словами и использовать Random чтобы сгенерировать
случайный индекс, по которому будет выбираться случайное слово из массива. Для всего этого действия будет удобно сделать
отдельный метод.

- Чтобы было легче работать, преобразуйте полученную строку в массив символов с помощью метода toCharArray(), который есть
в классе String (его писать не нужно, просто вызвать)

- Сделайте второй массив типа boolean, в котором будете отмечать, отгадана ли буква на определенной позиции. Очевидно, что
этот массив должен быть такого же размера как и массив с буквами.

- Создайте переменную, которая хранит количество попыток, изначально их должно быть 7. Сделайте цикл, который будет выводить
буквы и спрашивать у пользователя ввести букву. В массиве с отметками про отгаданные буквы, для соответствующих индексов букв в массиве с буквами
выставьте значение true если буква введенная пользователем есть в слове.
Если ни одна буква из массива букв не совпала с введенной пользователем буквой, отнимите одну попытку.

- Ваш главный цикл должен работать пока количество жизней больше нуля

- Чтобы прочитать одну букву (char), напишите scanner.next().charAt(0) либо scanner.next().toCharArray()[0]

Пример выполнения:
    Привет! Програма загадала слово! Отгадай его по одной букве за раз.
    Буквы:
    _ _ _ _
    Осталось попыток: 7
Ввод: e
    Буквы:
    _ _ _ _
    Осталось попыток: 6
Ввод: a
    Буквы:
    _ a _ a
    Осталось попыток: 6
Ввод: v
    Буквы:
    _ a v a
    Осталось попыток: 6
Ввод: k
    Буквы:
    _ a v a
    Осталось попыток: 5
Ввод: j
    Буквы:
    j a v a
    Осталось попыток: 5
    Поздравляю, вы отгадали слово!
```
