# Задание

* `Main01`
Сложность: легко (3/10)
```
Создать метод, который понимает, переменную какого примитивного типа данных ему передали.
Метод должен опозновать int, float, double, char и String. Также он должен выводить значение переменной.
В main привести несколько примеров вызова этого метода с разными аргументами.
Подсказка: использовать перегрузку методов

Пример кода:
    identifyType(1f);
    identifyType("Hello");
Пример вывода этого кода:
    Вы передали float! Значение: 1.0
    Вы передали String! Значение: Hello
```

* `Main02`
  Сложность: средне (6/10)
```
Написать метод, который спрашивает у пользователя целое число в нефиксированном диапазоне.
Метод должен переспрашивать, пока пользователь не введет подходящее число.
Диапазон (минимальное и максимальное число) должен задаваться с помощью параметров. 
Задавать этот диапазон нужно с помощью аргументов при вызове метода.
Использовать этот метод чтобы разработать программу, которая просит пин-код у пользователя, пока тот не введет правильный код.
Правильным кодом считаем 8956.
Подсказка: пин код это число из четырех цифр

Пример выполнения одной программы:
    Введите пин:
    7777
    Неправильный пин! Попробуйте еще раз:
    78
    Вы ввели неправильное число!
    123456
    Вы ввели неправильное число!
    5463
    Неправильный пин! Попробуйте еще раз:
    8956
    Пин код верный, доступ разрешен.
```

* `Main03`
  Сложность: средне (5/10)
```
Спросить у пользователя натуральное число от 1 до 50. Переспрашивать пока не будет введено правильное число.
Сгенерировать массив из случайных значений от -100 до 100, в качестве размера массива использовать введенное пользователем число.
Заменить все отрицательные значения в массиве на 0 и вывести массив в консоль.
Уточнение: в программе должно быть четыре метода:
1. Спрашивает у пользователя число
2. Генерирует массив случайных значений
3. Заменяет все отрицательные значения на 0
4. Выводит массив в консоль

Пример:
    Ввод:
    10
    Вывод:
    0 48 42 1 0 0 33 0 91 7
```
